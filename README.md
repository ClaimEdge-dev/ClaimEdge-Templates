## What's Included

This repository contains pure templates for ClaimEdge operations. All files use `[BRACKETED]` placeholders — no real claim data is included.

### Folder Structure

```
ClaimEdge-Templates/
├── invoices/              # Invoice templates (single, combined, day-rate, JSON schema)
├── emails/                # Email templates (5 stakeholder types)
├── ai-prompts/            # AI prompts for Claude, Perplexity, ChatGPT, Grok
├── rebuttals/             # 6 adjuster rebuttal scripts (phone + written)
├── xactimate/             # Xactimate line item reference
├── claim-packages/        # 12-document claim package checklist
├── partnership/           # Partnership proposal template
├── reference/             # Rate card, IRC codes, quick reference card
├── react-app/             # ClaimEdgeOS React invoice builder app
└── README.md              # This file
```

### How to Use

1. **Invoices:** Copy the appropriate template, replace all `[BRACKETED]` fields with claim data.
2. **Emails:** Use as starting points for contractor, adjuster, building dept, and homeowner communication.
3. **AI Prompts:** Fill in claim data and send to the appropriate AI tool for research or drafting.
4. **Rebuttals:** Reference during phone calls or copy written scripts into supplement letters.
5. **Xactimate:** Use line item codes when building estimates.
6. **Claim Packages:** Use the 12-document checklist to ensure every package is complete.
7. **Partnership:** Customize the proposal template for contractor negotiations.
8. **React App:** See `react-app/` for the ClaimEdgeOS Invoice Builder.
